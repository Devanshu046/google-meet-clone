# google-meet-clone
A real-time video conferencing application built with WebRTC.

## Tech Stack
- Angular
- Node.js
- Express
- Socket.io
- WebRTC

## Features
- Create meeting room
- Join using room ID
- Video and audio streaming
- Mute / unmute microphone
- Toggle camera
- chat
